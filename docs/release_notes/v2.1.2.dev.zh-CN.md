## 开发环境设置

### 要求
- Python 3.8+
- PyTorch 2.1.0+
- 开发工具
- 性能分析工具

### 安装
```bash
# 克隆仓库
git clone https://github.com/gomokuworld/gomoku-world.git

# 安装依赖
pip install -e ".[dev]"

# 安装性能分析工具
pip install -e ".[profiling]"
```

## 测试
```bash
# 运行所有测试
pytest

# 运行性能测试
pytest tests/performance/

# 运行AI测试
pytest tests/test_ai/
```

## 性能分析
```bash
# 分析AI性能
python tools/profile_ai.py

# 分析网络性能
python tools/profile_network.py
```
